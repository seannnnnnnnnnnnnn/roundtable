# Roundtable 0.1.0

首个公开版本。

## 核心能力

- 动态上下文补充问题
- 独立 Markdown 角色编制与选择
- 支持者、反对者独立首发和交叉回应
- 主持人提炼关键分歧
- 实践顾问给出可执行取舍
- 综合输出三层阅读路径
- 项目与会话本地隔离
- OpenAI-compatible Provider 配置
- macOS 桌面安装包
- GitHub Releases 自动检查、下载与重启更新

## 安装说明

请根据设备选择 `arm64` 或 `x64` 安装包。

如果当前构建尚未完成 Apple 公证，首次启动请在 Finder 中右键应用并选择“打开”。
